## Upload a video  - /uploads

<ul>
  <li>url:  /uploads</li>
<li>Method: POST </li>
<li>request body:  {file: File}</li> 
</ul>



## Get all videos   - /uploads

<ul>
  <li>url:  /uploads/ </li>
<li>Method: GET</li>

</ul>



## Get single videos   -  /uploads/:videoId

<ul>
  <li>url:  /uploads/:videoId </li>
<li>Method: GET</li>
</ul>
